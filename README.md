# Star Wars Character Searcher

A web application to search for Star Wars characters. Allows searching by name and viewing character details.

## Installation

1. Clone the repository.
2. Install dependencies:
   ```
   npm install
   ```

## Usage

To run in development mode:
```
npm run dev
```

This will start the development server at `http://localhost:5173` (or the port indicated by Vite).

To build for production:
```
npm run build
```

To preview the build:
```
npm run preview
```

## Technologies

- **Vite**: Fast build tool.
- **TypeScript**: For static typing.
- **Lit**: Library for web components.
